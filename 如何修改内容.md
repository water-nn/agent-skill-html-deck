# 如何修改内容

这个项目是网页式 PPT。你可以先只改内容文件，不需要理解整个代码项目。

## 新手最推荐只改这个文件

`src/slides.tsx`

这里放着每一页 PPT 的标题、分组、正文、卡片、标签和示例代码。

## 改标题在哪里

打开 `src/slides.tsx`，每一页都有这样的字段：

```ts
title: "Agent Skill 是什么？"
```

把引号里的文字改成你想要的新标题。

## 改每页正文在哪里

还是在 `src/slides.tsx`。每一页的 `content: (...)` 里面就是正文内容。

常见文字会出现在：

```tsx
<p>这里是正文</p>
<li>这里是列表项</li>
<Metric value="数字" label="说明" />
<Pill>标签文字</Pill>
```

只改中文文字最安全。

## 改按钮文案在哪里

按钮、目录、全屏、上一页、下一页等界面结构主要在：

`src/App.tsx`

例如：

- `Agenda`
- `Fullscreen`
- `Previous`
- `Next`

如果你不懂代码，建议先不要改 `src/App.tsx`。

## 改页数怎么做

每一页都在 `src/slides.tsx` 的 `slides` 数组里。

- 增加一页：复制一个 `{ ... }` 页面块，改 `id`、`section`、`kicker`、`title` 和 `content`
- 删除一页：删除对应的 `{ ... }` 页面块

注意：每页的 `id` 不能重复。

## 改颜色和风格在哪里

主要在：

`src/styles.css`

这里控制：

- 深色背景
- 动态光斑
- 卡片样式
- 按钮 hover 效果
- 全屏布局
- 打印 PDF 样式

如果你不懂代码，建议不要一开始就改这个文件。可以先告诉 Codex 你想换成什么风格，让 Codex 帮你改。

## 不懂代码的人应该改哪些文件

优先改：

1. `src/slides.tsx`
2. `如何修改内容.md`

暂时不要动：

1. `src/App.tsx`
2. `src/styles.css`
3. `package.json`
4. `vite.config.ts`
5. `preview.ps1`
6. `打开预览.cmd`

## 改完以后怎么看效果

双击项目文件夹里的：

`打开预览.cmd`

黑色窗口不要关。浏览器打开后就能看到改动效果。
