# 如何修改内容

这份网页式 PPT 的主要内容都集中在一个文件里：

`src/slides.tsx`

如果你不懂代码，建议先只改这个文件。其他文件能不动就不动。

## 改标题在哪里

打开 `src/slides.tsx`，找到每一页里的：

```ts
title: '这里是标题'
```

把引号里的文字改掉即可。

## 改每页正文在哪里

还是在 `src/slides.tsx`。

每一页都有 `content: (...)`，里面就是这一页的正文、卡片、列表或数字信息。

常见文字通常在这些地方：

```tsx
subtitle: '这里是副标题'
text="这里是卡片正文"
<li>这里是列表文字</li>
<p>这里是段落文字</p>
```

## 改按钮文案在哪里

翻页按钮、全屏按钮在：

`src/deck/Deck.tsx`

对应文字是：

- `上一页`
- `下一页`
- `全屏`
- `退出全屏`

如果你只是改 PPT 内容，不建议改这个文件。

## 改页数怎么做

打开 `src/slides.tsx`。

每一页都是 `slides` 数组里的一个大括号：

```tsx
{
  id: 'opening',
  section: '开场',
  title: 'Agent Skill 是什么？',
  content: (...)
}
```

想增加一页，就复制一整段类似结构，粘贴到 `slides` 数组里，并改：

- `id`：每页必须不同，只用英文、数字、短横线
- `section`：左侧目录里的分类
- `eyebrow`：页面上方的小标签
- `title`：主标题
- `subtitle`：副标题
- `content`：正文区域

想删除一页，就删除对应的大括号段落。注意不要漏掉逗号。

## 改颜色/风格在哪里

主要在：

`src/styles.css`

常见颜色关键词：

- `cyanGlow`：蓝绿色高光
- `violetGlow`：紫色高光
- `aurora`：绿色高光
- `#050812`：深色背景

如果你不懂 CSS，建议先不要改 `src/styles.css`。

## 新手建议只改哪些文件

优先只改：

`src/slides.tsx`

可以阅读但不建议随便改：

- `如何修改内容.md`
- `部署说明.md`

不懂代码时尽量不要改：

- `src/App.tsx`
- `src/deck/Deck.tsx`
- `src/deck/SlideFrame.tsx`
- `src/deck/hooks/*`
- `src/styles.css`
- `package.json`
- `vite.config.ts`

## 改完以后怎么看效果

双击：

`打开预览.cmd`

黑色窗口不要关。浏览器会自动打开预览页面。
